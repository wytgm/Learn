                                # 本书情况介绍

                                《艺术学概论》主要有五部分：

                                • 艺术本质论
                                • 艺术创作论
                                • 艺术作品论
                                • 艺术鉴赏论
                                • 艺术发展论

                                市面上有四个版本，彭吉象、王宏健、王一川以及马工程，山师考研22年以前是彭吉象版的，现在用的是，可以结合使用。

                                山师广电的题型是完全固定的，考察的题型也都是比较常见的 需要在学习过程中积累一些影视作品。从近几年的出题情况来看，影视作品的出题都比较常规，并且有迹可循，所以这个题不用太担心不会的情况。

                                山师不考察选择填空这样的小题型，所以更加需要从整体上进行把握专业课本，并且有一个需要注意的点是，336题目并不难，，基本上背的熟练都可以答上去，但是要注意把握答题时间，从往年的考试情况来说，每年都有学生答不完题，甚至到最后剩下25分的论述题来不及写。

                                所以备考时要注意两个问题：

                                第一，专业一艺术基础只有一本课本，所以因为只有对题目足够熟悉，才能减少思考时间，迅速作答。

                                第二：后期一定要做真题训练，才能保证在考场上把自己会的题都答好。

                                # 技巧

                                名词解释：300字左右

                                解释 +（人物 + 人物代表的观点）+ 举例 + 自己的观点。

                                简答：不少于500字，

                                书中内容 + 举例说明（至少两个）

                                论述：1000字打底

                                大概率是影评....整理成自己的模板【待整理】

                                这几年考的导演来看，常考第五代导演：“冯小刚、陈凯歌
                                张艺谋”，港台导演王家卫、第六代领军人物贾樟柯。

                                电影批评的方法有很多，借助一个批评理论整理好自己的影评模板，这样才能拔高自己的影评高度，例如，从接受美学角度出发，那就要去了解其理论的具体内容，“空白”、“不确定性”、“期待视野”等，都可以成为影评的论点

                                总之，能编就使劲编，千万别空着，最好用自己的课外例子，
