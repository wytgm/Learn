                                      # 第九章 艺术批评

                                      这个章节也真的很重要📢，包含了艺术批评所有的知识点。

                                      # 第一节 艺术批评简述

                                      # 🌻 一、艺术批评的含义与性质 名词解释

                                      艺术批评是遵循相应的批评原则和艺术标准，对艺术作品，艺术家，艺术现象等进行理性分析和科学评价，在艺术生产与文化建设中发挥着重要的功能，具有重要的意义。艺术批评应坚持美学的观点和历史的观点及美学标准和历史标准相结合的原则，以及“真、善、美”的统一标准。艺术批评可以影响并指导提升艺术家的创作影响，并引导艺术接受者的艺术接受活动，更具有重要的文化传播与文化建设的意义。我们提倡并实践一种在克思主义文艺理论思想指导下的坚持民族文化自信，同时广泛借鉴、批判继承汲取的，“开放的艺术批评” 总之艺术批评是艺术鉴赏的升华，他可以指导艺术鉴赏和艺术消费，同时对艺术创作具有指导意义，也是艺术理论的实践，是療史的有机组成。进入新时代，在习近平文艺理论的引戢，艺术批评更为活跃，将在推进文化建设中发挥重大作用。

                                      # 二、艺术批评的原则 简答

                                      艺术批评虽然有批评家的主观性介人和主体性差异等特性，但同时也具有客观的原则和标准。艺术批评的标准因批评者对艺术本质、艺术功能等理解的不同而不同，艺术批评的标准。定程度上也是时代、民族、文化等的要求的体现。

                                      第一个是恩格斯主张的美学的观点和历史的观点及美学标准和历史标准的结合。这一原则要求我们在评价艺术作品艺术性实际，要从美学的角度对艺术作品进行艺术的分析。还要以发展的历史的眼光来看待艺术作品。以电影史为例，即便有些老电影在今日看来不无简陋和粗糙，但它在某些方面的“经典性”就像原始神话的独特魅力那样，并非单由技术、技巧所能超越。

                                      第二个重要批评原则是“真、善、美”的统一标准，也即形式与内容相结合的标准或原则。“真”是对艺术作品真实性的要求;“善”是对艺术作品要符合社会核心价值伦理观的要求，“美”是关于艺术作品美学艺术方面的要求。这三者形成一种互相依存的关系，不能片面割裂。但无论是真善美还是内容与形式的完美结合，整体的和谐统则是必然的要求。

                                      除了这两个重要的艺术批评原则，毛泽东提出的“政治标准”“艺术标准"相结合,习近平提出的文艺作品“艺术性”，“思想性”，"价值取向”相统一，也是重要的关于艺术批评原则或标准的思想。体现了马克思主义文艺观的中国化，时代化的重要成果。

                                      # 三、艺术批评的意义 简答

                                      艺术批评是依据一定的思想立场和美学原则，对艺术作品，艺术现象，艺术家等进行理性分析和科学评价的艺术活动。习近平在文艺工作座谈会讲话时便指出：“文艺批评是文艺创作的一面镜子，一剂良药，是引导创作多出精品，提高审美，引领风少的重要力量，显然，文艺批评具有十分重要的意义与作用。” 艺术批评的功能或意义主要有三个方面：

                                      1、通过把阐释、解读及批评的信息反馈给艺术家，影响并指导、提升艺术家的创作。艺术创作是一种非常复杂的精神生产活动,艺术家需要来自受众专家的批评反馈，而进一步加深对自己创作的认识。例如中国电影三大奖，就是在不断地激励中国影视人制作水平的不断提高。

                                      2、通过对作品的分析和阐释，评判其美学特征、艺术价值，介入艺术接受活动，从而影响并引导接受者的艺术接受活动，帮助受众更好地鉴赏艺术作品。也能在引导艺术市场及受众艺术消费等方面起到一定的作用。例如，对于电影的接受来说，口碑已经成为影响票房的很重要的因素。但是值得注意的是，艺术批评还能指导受众规避不良艺术作品可能产生的负面影响，使受众形成警惕意识和批判思考的精神。

                                      3、艺术批评更具有重要的文化传播与文化建设的意义。一方面,它通过对艺术创作实践和艺术交流活动的积极介入发言和引导,成为艺术传播和文化传播的有机一环。它能够影响艺采传播的速度，广度和方向以及效果，也可以协调艺术与社会意识形态之间的关系，参与艺术市场的调节和社会文化建设。例如好的艺术批评能够营造良好和谐的文明鉴赏环境，提并国民的文化艺术修养与精神素质。

                                      另一方面，艺术批评的这种传播性，能够使艺术作品实现它其真正的意义，艺术交流是通过艺术家所创造出来的符号化的艺术作品而进行的，这种艺术活动的社会特征和艺术的符号性决定了艺术传播的必然性，艺术作品的功能、价值、意义才能得以实现，传播媒介与方式的改变又会反过来影响艺术的功能、意义与价值。

                                      # 第二节 艺术批评的主体与主体构成

                                      # 一、艺术批评的主体 简答

                                      艺术批评是由艺术批评者主体进行的感性与理性相结合的实践活动。艺术批评也是批评者主体力量的对象化。从艺术生产的角度看，艺术批评也是艺术创作或艺术生产的有机组成部分。艺术批评则更多是分析、研究、判断的理性思维活动。

                                      其一，艺术批评家主体需要具备较敏锐的艺术感受力，能对艺术作品、艺术形象的艺术特质，美或丑，和谐或破碎，形式与内容等有准确的把握，有独具慧眼的发现。郭沫若认为，批评是发现的事业。是在砂之中寻出金。

                                      其二，艺术批评家应该具备深厚的哲学、美学修养、知识，以及开阔的文化视野，尤其应该有深厚的传统文化素养，以及充分的文化自信。在艺术史上，许多批评大家几乎同时也是哲学家和美学家，如黑格尔、克罗齐、马克思。

                                      其三，艺术批评家应该具有艺术史、艺术理论的专业知识，要懂得艺术语言，最好还要有一定的艺术实践，在此基础上将会具有更好的理性思考和理论概括、提升的能力。如美学家宗白华也是批评家，他对中西艺术的深厚学养，使得他能够在各个艺术种类如美术、音乐、戏曲、舞蹈、书法、建筑中举重若轻。

                                      其四，艺术批评家还应有良好的符合社会要求的伦理道德索养，特别是要具有独立的品格，不迎合世俗，更要拒绝所谓的无原则、无艺术标准的“红包批评”。

                                      # 二、艺术批评的主体构成

                                      艺术批评主体即艺术批评家或艺术批评者，大致由专业(学院)批评家、艺术家批评家、媒体批评家和大众批评家组成。专业批评家大多受过艺术批评的专业训练，有扎实的艺术素养以及哲学、美学、艺术学的理论素养。艺术家批评家是指主要身份是艺术家，但偶或从事艺术批评，阐发自己的艺术观、创作思想，或解释自己的艺术作品等。

                                      在一个多媒介的时代，媒体批评者与网民批评者颇有崛起之势。媒体批评者主要指为各种新闻媒体进行新闻性艺术报道的记者式的批评者。20 世纪在美术领域还有一个重要的艺术传播方式或艺术批评媒介即美术展览。此时批评家的角色又表现为策展人。还有非职业性的、凭自己的兴趣进行批评的群众、网民批评者群体。

                                      # 第三节 艺术批评的维度与方法

                                      # 一、艺术批评的维度或角度

                                      鉴于艺术的复杂性切入对艺术的思考和批评的角度无疑有很多。

                                      （一）世界或现实维度

                                      从这一角度切人的艺术批评一般是社会文化系统的批评，重点关注艺术表现世界与现实对象世界的关系。19 世纪法国文学批评家丹纳在《艺术哲学》中提出了决定文艺的“三要素”说，这“三要素”即种族、环境和时代。从这一角度切入对艺术的批评主要是把艺术作,晶看做是对社会文化、社会意识形态的一种反应或再现，其艺术标准一般侧重于艺术表现的真实性问题，推崇 现实主义纪实风格的艺术作品。力图把艺术看作是一种开放性的，与市场经济， 工业生产方式与社会意识形态的再生产密切相关的文化现象，从而进行的文化研究和文化批评。

                                      （二） 创作主体纬度：艺术创作或生产系统

                                      从这一角度，可以对艺术家主体创作或生产的各个环节进行批评，戏剧与电影艺术具有综合性，可以从这个角度对编剧，导演，表演，摄影，美术等进行批评研究。还可以对艺术创作主体进行研究，包括构成导演风格构成的几种要素，如题材的选择，剧本的结构，物台或画面的造型，以及演员表演，音乐，音响等辅助元素的运用。另外也可以对艺术创作主体进行创作心理学的研究，研究作品与艺术家的生平，个人经历和童年记忆等方面的关系。例如导演贾樟柯，出生于山西汾阳，他在县城生活和在京城求学以追求自我价值实现的过程中，感知了历史转型时期人们的迷茫、困惑和无所适从。这给了他很大的影响，以至于成为导演以后，他开始以电影为媒介来考察当代都市普通或边缘人的生活状态，用镜头语言讲述人在时代浪潮中的变化，关注着生命的卑微与伟大。从而形成了自己独特的风格。

                                      （三） 语言与形式纬度：艺术本体系统

                                      从这一维度进行的是一种注重艺术语言本体、侧重形式的文本批评。所谓“文本批评”就是视艺术作品为表意的表述体，进而分析研究这一表述体的内在系统构成和表意的方式。例如，就一部影片而言，我们可以研究影片的语言特点，如画面造型的特点，运动镜头，语言，剪辑风格，叙事或抒情的节奏等内容，我们也可以对电影结构影片表面化的主题立意进行解读。

                                      （四）心里接受纬度：受众系统

                                      从这一角度可以研究受众接受、窥视、文化认同等心理，研究艺术对受众心理的自觉或不自觉的投合。在艺术鉴赏过程中，研究作品对观众的“缝合”作用及过程，也可以进行原型心理研究或进行个体精神分析，进而研究作品中所表的大众文化心理。

                                      # 二、艺术批评的方法（简单背诵）

                                      所谓艺术批评的方法，是指对艺术作品及一切艺术活动艺术现象予 以理性分析，评价和判断的方法。艺术批评的方法是丰富多样的，且常常是复合的，可以说并不存在绝对单一的批评方法，对一部作品以为艺术家，一个艺术家群体或艺术现象进行的批评可能有角度的不同方法的侧重等。但实际上，在批评实践的过程中，必然会综合各种批评方法，批评方法的区分，只是为了理论分析的需要。

                                      （一）偏重艺术作品的表现对象即世界本体的批评方法

                                      代表人物有亚里士多德，擀尼雪夫斯基。这是一种被称为“社会历史批评” 的方法，这种批评方法从旳历史的角度来芬析艺术作品，研究艺术与社会，历史，人生的关系。一般而声，他主张女术与社会，艇,人生有着密切的近乎反映和再现的关系，重视艺术的社会功能和作用，其评夠尺度主要是真实性，倾向性和社会效果。

                                      该观点在西方学认为，艺术是对客观世界或社会生活的真实的模仿和再现，艺术家对现实的描绘越逼真，其艺术价值就愈高。在中国古代的文艺理论中，有一种“实用论”也是强调文艺的社会功能，这一批评传统强调文艺批评的社会、伦理、道德标准和人生、社会结合起来，坚持用道德的观点来看待文学现象。

                                      （二） 偏重艺术创作主体的批评方法

                                      艺术家主体在艺术中占着重要的位置，艺术批评以艺术家主体为主要对象的批评大致可归纳为一种主体批评。艺术家个体的心理，性格，趣味等特征会明显的烙印在他的作品之中。

                                      这一批评方法认为，艺术是艺术家心灵世界的表现，是艺术家情感的自然流露，因而主张艺术的本源来自于艺术家的心灵状态，一件艺术品本质上是内心世界的外化。

                                      （三） 偏重于作品本身的批评方法

                                      这是 20 世纪崛起的形式主义批评潮流，表现出明显的重形式，轻内容的形式论倾向。他们也承认有形式内容之区分，但认为内容不属于艺术理论研究之列。艺术理论研究的是进入艺术作品的内容一一形式。偏重作品形式结构等的批评，是一种注重对艺术作品本身进行的本体研究，对于在其之前的社会历史评判（重世界、重模仿或反映）及浪漫主义批评（重创作主体、重表现）的矫正。当然，形式主义倾向的艺术作品本体研究，细致、严谨有余，灵活不足，常馅于机械、呆板、孤立、封闭的僵局。“一叶障目、不见泰山。”

                                      （四） 偏重于接受、受众心理的批评方法

                                      严格意义上来说，缺少艺术接受这一环节，艺术还是不完整也是不存在的。真正只有为自己创作的艺术家是没有的，真正停留在创作完结阶段的艺术作品也是不存在的。20 世纪以德国的姚斯，尹塞尔为代表的接受美学和以伽达默尔为代表的阐释学都称得上是走向“接受之维”的艺术理论。

                                      这一批评方法还突出艺术文本的未定性。或艺术文本的“空白”，主张空白是交流的基础，是接受者产生积极接受和阐释的动力，没有空白就没有交流的必要。强调文学史的接受因素，因为文本的未定性不仅存在于文本结构之中，也存在于历史之中。因为接受者是在艺术史的序列中，在对艺术史的认知中建构对艺术对象的判断与认知艺术作品的社会意义，审美价值都需要在接受者的创造性接受活动中才得以实现。

                                      （五）其他具有一定的综合性、多维度特征的批评方法

                                      神话原型批评方法

                                      神话原型批评方法认为，许多优秀艺术作品感人的深层动因在于其叙事结构或人物命运中所隐含着的原型，正是原型引发观众深深的共鸣。

                                      意识形态批评方法

                                      意识形态是马克思主义文艺观的一个核心概念。意识形态也是文化批评的一个核心性的范畴。所谓意识形态，一是指二种支配个人心理及社会集团心里的。反映某一个阶层，集团，阶城文化群体的社会要求和理想的一整套的思想体系。因此，艺术家离不开意识形态的控制和反控制，需要在限制中求自由。

                                      艺术作为一种特殊的意退幟，是对社会，人生的矛盾做出想象性的解决。如电影，不论他的商业动机和美学要求是'杵么，电影的主要魅力和社会文化功能 基本上是属于意识形态的电影，实际上是在协助公众去界定那些迅速演变的社会 现实并找到它的意义。

                                      总的来说，我们的批评方法和原则应该是一种综合的，灵活的，超越性 的研究。在批评方法的实践中，还需要致力于中华传统优秀艺术理论，批评资源以及中华美学精神的阐发，从根本上说，我们应该提倡一种在马克思主义文艺理论思想指导下的“开放的艺术批评”的立场，他是对现有各种批评方法的综合, 是对中华传统文化美学精神批评原则的综合与现代化，也是对现有所有单一的批评立场的超越。
