                                      # 第十章 艺术的当代嬗变

                                      在世界经济全球化的背景下，当代艺术呈现丰富的多元文化特征。同时，各个国家和地区积极发展本土艺术的同时，也积极参与艺术的交流与对话 不同文化特质的艺术共存互动现象更加凸显，艺术与社会的联系愈加紧密，创意产业和艺术市场不断壮大，文化消费中的高雅艺术和大众艺术受到关注。以数字媒体技术为核心的媒介变革在当代艺术嬪变中的作用越来越大，推动了艺术各门类在内容形式和审美特征上的发展变化。

                                      # 一、 艺术中的多样化与本土化 简答

                                      （简单背诵 总分总）

                                      世界各国政治经济科技，文化信息日益紧密的相互联系和彼此交融， 使艺术成为人类社会科学中的一门高度学科化的知识体系。全球化促成了不同文化和文明间的对话，有利于艺术的创新与发展。也有利于人们在对文化“他者”的审视中，加深对本土艺术个性的理解，促进艺术观念与形态的多元化。多样化与本土化的艺术博弈也带来了需要思考的问题，问题如下：

                                      首先，多样化时代的各国艺术出现了,明显的国际化现象。 由于人们获取艺术信息的数量和内容高度凋，跨国艺术体制下的艺术理论，收藏批评和市场紧密联系，相关的艺术批评从过土的审美讨初更多的转向文化批评，激发了多元视角下的文化民族自觉，艺术的本土化成为一个重要的议题。 例如，米兰时装秀中的服装文化很快就会在全世界流行，成为国际服装文化。

                                      其次，艺术的本土化在国家和民族文化建设中的地位与作用不断加强。 在特殊的历史情境下，提倡与推广本土艺术风格不仅仅是艺术本体语言和审美意味的文脉继承，也是提高国家和人民的文化认同感。组织动员文化资源，实施文化遗产保护，国家软实力建设，以及强化本民族或国家形象认同和集体归属感的需要。例如，一首义勇军进行曲激发起中国人民抵抗外敌侵略的勇气和决心。

                                      最后，本土化是创造艺术现象的根本语境。 一篇散文，一首诗，一部电影，一曲音乐，都能给外国人了解中国提供一个独特的视角。文艺工作者要讲好中国故事、传播好中国声音、阐发中国精神、展现中国风貌，让外国民众通过欣赏中国作家艺术家的作品来深化对中国的认识、增进对中国的了解。

                                      当代艺术的本土化需要构建与之相对应的艺术理论体系，以作为艺术质量评判和内容阐释的支撑。以中国当代艺术为例，如果缺乏本土理论系统的构建与阐释，缺乏对中国当代社会语境下中国本土思想与文化的艺术转化与意义阐释，本土化只能是一种理想而不会成为现实。同时也要避免将本土化作为简单复活国粹或传统的一种形式。而是要在正式和吸收当下的多元与融合的同时，继承和发扬自身文化艺术传统的精髓。

                                      # 二、 高雅艺术与大众艺术

                                      现代艺术的发展加强了两种类型的文化对比：高雅艺术与通俗艺术，精英艺术与大众艺术。高雅艺术是高端，精致，典雅的艺术。所谓的高雅艺术多指小众精英人群才能欣赏到的“阳春白雪”的艺术，也是指经典的艺术，其特点是艺术品质高超，主体思想高超，具有保存和流传的经典型。其可以涵盖京剧，话剧，交响乐，歌剧，芭蕾舞，国画等。大众艺术是指为群众所喜闻乐见的艺术，是被占社会人口绝大多数的民众接受，参与，欣赏消费的艺术。它的最重要的特征是受众群体的庞大数量。大众艺术涉及流行艺术，商业艺术，群众艺术，其根源在于大众阶层的崛起。

                                      在大多时候，人们讨论艺术的雅俗问题并无高低。尊卑之意，而是指两种不同的艺术风格与品格，雅俗共赏，也是不少优秀作品的特征。对于高雅艺术与大众艺术来说，人类精神中有对高尚、崇高的向往，同时也是有对休闲、娱乐的要求。大众艺术可以使高雅艺术雅俗共赏，是古今中外许多优秀艺术作品的特征本质，无论是高雅还是大众的艺术，都需要避免低俗，追求好的品质。对大众艺术通俗性和小众艺术实验性，先锋性的包容，有助于形成和谐的社会文明，有助于推动社会主义艺术事业的发展。

                                      🌻 补充：如何看待“雅”和“俗" 简答

                                      雅和俗是常常被提及又容易被混乱使用的概念，特别是对俗的理解，在某些艺术家眼里被解读为真实。俗的本意是指日常习俗，风俗是社会上流行的生活习惯，后来渐渐变成纸。档次不高的情趣与品位在艺术上多形容庸俗且带有感官和功利色彩的粗糙的艺术，贬义的俗的艺术，似乎是凭着肤浅的市井经验和趋利避害的直觉就能生成的艺术。它的对应概念则是经过学习才能被小部分人掌握和理解的精致的雅的艺术，是一种具有艺术品质高超，主体思想高尚，具有保存和流传的经典性的艺术。人们在讨论雅与俗的问题时。其实并无高低悲尊之意，而是指两种不同的艺术风格与品格，雅俗共赏是不少优秀艺术作品的特征。

                                      🌻 补充：大众艺术 名词解释

                                      大众艺术， 特别是在中国革命和建设时期形成的大众艺术是服务于群众，为群众所喜闻乐见的艺术，是被占社会人口绝大多数的民众接受、参与、欣赏、消费的艺术。它的重要特征是受众群体的庞大数量。在现代社会中，特别是在消费文化语境中，大众艺术呈现出一 些鲜明的特征。

                                      # 第二节 创意产业和艺术市场

                                      经济产业助推创意转化成产品生产出价值。艺术市场，实现艺术商 品的交换，满足消费者的艺术体验与精神需求，提高生活品质。文化创意产业和艺术市场是社会经济的重要组成部分。在促进艺术生产的同时，也承担着创造精神价值，产生社会效益的责任。

                                      # 一、文化创意产业发展现状/特点

                                      发展文化创意产业，是社会主义市场经济条件下满足人民多样化精神文化需求的重要途径。文化创意产业是艺术，在全球化背景下，根据社会政治和经济发展需要，以创意和创造力作为核心经济生产要素而发展起来的新兴产业形式。文化创意产业是从传统文化产业发展出来的新型产业，它的核心是“创造力”和产业经济的结合，文化创意产业有以下几个特点：

                                      1、通常由国家和地区政府利用文化制度和管理资源，配套以法律，资金等 扶持手段，提高本国和地区文化与经济的国际竞争力。 当代文化创意产业体现出更加显著的国家战略和地域特征。例如，日本借助其游戏动漫产业的领先优势，特别强化内容产业的提法益，突出数字产品文化内容创新的重要性。

                                      2、经济全球化和文化本土化是文化创意产业发展的重要基础。 一方面，国家文化竞争力和软实力通过文化产品的生产与输出从而得到体现。另一方面，强大的跨国资本形成了成熟的全球生产体系。文化创意产业的生产组织和链条也是多样国际化的。但与此同时，由于各国地区存在文化消费习惯的特殊性，全球化业同时面临着国家间文化竞争的博弈问题。

                                      3、新技术和新媒介对文化创意产业的发展起着极为重要的支撑作用。 文化和科技在日新月异的技术革新基础上产生了不可估量的化合作用。内容的传播与交互方式的创新，资本与平台资源的结合，都会因为技术进步而引发新的变革。正如网络宽带和手机移动端技术提升引发的数字内容革命，技术创新改变的不仅仅是产业形态，更是人的生活方式。

                                      4、文化创意产业具有典型的城市文化特点。 文化消费，艺术消费，娱乐消费是人类社会城市化进程中普遍进入泛娱乐化时代的表征。例如以艺术，影视娱乐，体育旅游为主的美国休闲文化产业就已经成为国民经济的支柱产业。

                                      5、具有鲜明的产业链形态特征。 文化产品的价值产生具有鲜明的阶段属性。以创意为龙头，以内容为核心，以设计为形式，通过产品的制造与营销建立和流通传播渠道。依靠适当的发行平台进行后续的衍生产品的开发。形成“上下联动、左右衔接、一次投入、多次产出”的产业链。

                                      6、跨界合作成为文化创意产业的常态及趋势。 文化创意产业需要将艺术感觉, 文化在身，技术创新和投资管理渠道等资源进行整合，于是附和于跨界性的合作就成为了让创意变成产品，最终成为好产品的建设性方式。

                                      🌻 政府在文化创意产业中应当扮演怎样的角色？简答

                                      资助人： 在国家文化制度和管理资源中，政府在文化创意产业中应当扮演资助人的角色，进行政策及资金的扶持。例如，在国外就有很好的例子。美国政府对公共项目中艺术品的投痛有一定比例的要求，疝建设大型建筑需要专设一定比例的资金用于艺术品的装饰，对自助公共艺术的个人和企业也有相应的税收优惠政策。

                                      管理人: 政府还应该营造城市创意文化和产业环境。城市管理者通过文化创意产业园区的设立，加强区域文化产业的竞争力，产生良好的聚合效应，引入可以产生化合作用的优质文化业态群，避免创意园区变为一般房地产项目。这样的话，便可以促进就业和可持续发展。

                                      建设人: 一个城市的创意环境建设首先是创意人才队伍的建设。因此，政府也应该是一个建设人的角色，为此应该高度重视创意环境的生态建设，自然景观，历史文化和有故事性的环境可以激发文化内容的创新成果。

                                      # 二、艺术市场与艺术营销

                                      🌻 艺术市场 名词解释：是艺术品交换的领域和场所，广义的艺术市场指的是整个社会有关艺术品交换，流通，消费的供求服务关系体系，包括与 艺术品版权交换和艺术品交易所涉及的各类法律、法规、制度及相关软硬件条件。狭义的艺术市场是指直接开展艺术品交易服务的具体场所，在网络信息时代，既包括实体性线下卖场，也包括虚拟在线的艺术电商。

                                      🌻 艺术营销 名词解释：是以获取效益为目的，围绕交换进行的一系列市场策略及行为营销的最终目的是有一系列市场新活动产生出一定经济效益或其他收益。

                                      艺术市场和艺术营销具有与其他类型商品市场交换和市场营销不尽相同的特点。

                                      艺术商品具有文化与商业的特殊双重属性。艺术商品一方面能够满足人们对精神、心理，以及感官体验的审美需求，另一方面，艺术商品还具有交易，投资，分配，流通等商业属性。艺术商品有特定的定价、评估机制。艺术商品价值估价更重要的影响因素来自于复杂的心理需求与社会文化因素，需要考虑其作品类型、物理与媒介状态、 历史定位于学术评价、现实世界的使用功能等等的一系列因素。

                                      艺术市场具有鲜明的层级特征。不同体验类型的艺术消费形态，其差别是巨大的，由此产生了不同的市场营销对策。例如在特殊时空条件下及时进行的体验性消费，如游戏，休闲，娱乐，演出等，这类型就是通过消费者购买陪伴时间的体验来实现其价值，更多的与体验效果、媒体平台、传播方式、渠道和技术产生关联。

                                      价格能否代表艺术作品的价值是一个有争议的问题。通常认为，价格仅仅反映作品的市场价值，不能代表艺术的审美和精神价值。但在市场交换中，某些艺术的商品价值的确是通过价格来体现的。

                                      # 第三节 数字条件下的艺术

                                      数字技术条件下的当代艺术嬉变的一些共性特征：

                                      数字媒体使信息传播成为艺术本体的核心问题。社会心的交流需要和自我认同，成为启发艺术生产，推动艺术交流的重要原邕。艺术不再局限于单一的形态的作品。跨媒介，新材料的综合艺术不断地涌叫，数字技术所阐发的信息解码和传播方式被认知和被感知的方式交织成新的艺术形态和观念构成规律，使当代艺术语言与交流媒介更趋于销富与敏感。

                                      科学与艺术，技术题巧的融合改变了传统芨术的创作环境和风格特征， 数字技术介入传统艺术之后，个人创作风格于手工艺。关系逐渐弱化，观念的创新性体验的新奇感逐渐加强。不同风格的艺术家不再仅仅以手法和技巧相区别，取而代之的更是问题意识和态度。

                                      艺术的社会性与公共性显著加强，数字技术创造了新的公共艺术景观，让艺术打破传统媒体的单向和模糊图像，更清晰生动的艺术景观呈现在社会、大众和公共的空间中。艺术信息传播的交互性不断加强，艺术与社会公众的关系愈加密切。

                                      # 四、数字技术条件下的综合艺术

                                      艺术界定与边缘的综合性与模糊性不断加强,综合艺术可以任何几种媒介形态的综合形式而存在，图片，声音，影像，互动，游戏和表演均可共存共生，其核心是唤起有意味的体验。

                                      数字技术介入艺术创作带来的感官奇景。从早期略显粗糙的三维仿真影像, 到现在完全乱真的影像特效，数字技术在影像虚拟与视听效果上的巨大进步，已经将观众的感官刺激胃口大大提升。例如《泰坦尼克号》和《阿凡达》。

                                      数字媒体技术的进步对影视艺术的产业形态产生了重要的影响。随着 3D 数字电影技术的成熟和剧目影院的普及，观众又回到了影院，争先恐后的参与，新奇的视听体验，技术环境，如此明显的改变了产业形态。

                                      影视文化传播的巨大变化。数字媒体制造了全民娱乐的影像文化，不同文化背景的个体与群体文化立场的视角的多元化，影视语言表达的个性化，数字化, 社会化传播的迅捷与便利，让影视文化审美多样性，大众性碎片化，娱乐化愈加明显。

                                      # 第四节 当代媒介与艺术传播

                                      一、纸媒介与艺术传播（略）

                                      二、电子媒介与艺术传播

                                      在艺术传播发展中，电子媒介解决了传播的速度和效率难题，实现了声音和影像的实时传递，以摄影，广播，电影和电视技术为代表的电子媒介诉诸视觉和听觉革命成为工业时代占据主导性的艺术传播方式。

                                      尽管电子媒介对大众具有惊人的传播效率和视听优势，却由于受众参与程度低，代入感较弱，难以让大众真正享受到参与和分享的体验。

                                      三、网络媒介与艺术传播

                                      网络媒介解决了电子媒介无法达到的传播互动性，海量性和虚拟仿真性特征, 创造了沉浸式体验条件，是数字时代最为重要的艺术传播方式之一。

                                      具有强大的交互性

                                      具有沉浸式体验优势

                                      促进多屏媒介的大规模发展

                                      具有多媒体，跨媒体以及融媒体的特征

                                      （总）依托网络媒介发展起来的互联网艺术，微艺术，微拍等社会媒体和自媒体的迅速发展，给年轻艺术家创作了更多展示传播的平台。网络媒介对艺术传播的影响说明传播技术进步对艺术发展的作用并非可有可无，网络时代的艺术传播和文化交流对艺术的传播属性和媒介功能提出了新的要求，艺术的传播媒介属性和公共媒体属性不断加强，当代的艺术真正进入一个崭新的认识共享与创造的时代。
