                                                            # 第四章 运动摄像

                                                            运动画面：通过摄像拍摄方式所拍到的画面，成为运动画面。
                                                            运动摄像形式：推拉摇移跟升降以及综合运动摄像

                                                            # 第一节 推摄（推镜头）

                                                            推摄是摄像机向被摄主体的方向推进，或者变动镜头焦距使画面框架由远而近向被摄主体不断接近的拍摄方法。用这种方式拍摄的运动画面，称为推镜头。

                                                            # 推镜头特征

                                                            ①推镜头形成视觉前移效果。

                                                            拍摄时，镜头向前推进的过程造成了画面框架的向前运动，从屏幕画面上看，画面向被摄主体方向接近，表现出一种逐渐接近被摄主体的视点前移效果；同时形成了一种较大的景别向较小景别连续递进的过程，具有大景别转换成小景别的各种特点。比如，用推镜头拍摄一位被访者，在一个镜头里画面，从人物的全景环境到人物的面部特写景别变化“一气呵成”，观众从画面中看到了被访者从环境到人物表情的景别连续变化的过程。

                                                            ②推镜头具有明确的主体目标。

                                                            推镜头拍摄时，都具有起幅、推进、落幅三个部分。推镜头画面的向前运动是具有明确推进方向和终止目标的，这个终止目标既是最终所要强调和表现的被摄主体，由这个主体决定镜头的推进方向和最后的落点。比如在拍摄奥运会中运动员领奖牌的画面，镜头由全景一直推到运动员的特写，全景是起幅，特写是落幅，在起幅与落幅之间连续的画面运动即为推进。

                                                            ③推镜头将被摄主体由小变大，周围环境由大变小。

                                                            随着镜头向前推进，被摄主体在画面中由小见大，由不甚清晰到逐渐清晰，由占画面比例较小到占画面比例较大，甚至充满画面。与此同时，主体周围所处的环境由大到小，由占较大的画面空间逐渐变成占越来越小的空间，逐渐消失“出画”。比如在电视剧《大秦帝国》中，商鞅来到六国交战的战场辅佐恩师。

                                                            # 推镜头功用和表现力

                                                            ①突出主体人物，突出重点形象。 推镜头在形式上通过画面框架向被摄主体的接近，从两个方面规范了观众的视点和视线：一方面，镜头向前运动的方向性具有“引导”甚至“强迫”观众注意被摄主体的作用；另一方面，推镜头的落幅画面最后使被摄主体处于画面中醒目的结构中心位置，给人以鲜明强烈的视觉印象。在拍摄新闻场面时，推镜头唱被用来选择和交代众多参与者中的新闻人物、重要人物、领导者或权威人士等。

                                                            ②突出细节，突出重点的情节因素。 推镜头能够从一个较大的画面范围和视域空间起动，逐渐向前接近这一画面和空间中的某个细节形象，使这一细节形象的视觉信号由弱到强，并通过这种运动所带来的变化引导观众注意到这一细节。如在《法治进行时》栏目“人去楼空的补习班”一期节目中，当记者追问留守人补习班负责人的联系方式时，他称自己手机没电了，敏锐的摄像记者从全景画面推到手机的特写画面，手机画面却在频频闪烁。

                                                            ③在一个镜头中介绍整体与局部、客观环境与主体人物的关系。 推镜头从环境出发，通过镜头运动进一步深入到该环境中的人物，因而在一个镜头中就能够既介绍环境由表现特定环境中的人物。在影视剧中，这种方式也时常被用来表现特定环境中的特定人物。美国影片《指环王》中有这样的情节：护戒小分队走投无路，投奔人王，他们来到人王的大殿，随着镜头推上去，观众看到：人王面目黑暗、脸色阴沉，连之前看到的光亮这会儿也露出了真正面目，是象征邪恶的蓝光。另外，推镜头从环境到人物的表现方式还有强调全局中有这么一个局部的意味。比如，镜头从教室的全景推至学生小王，画面语言表达了“教室里有小王”的意思，强调了“小王”这一重点形象。

                                                            ④推镜头在一个镜头中景别不断发生变化，有连续前进式蒙太奇句子的作用。 推镜头是画面空间从小到大，向前推进。它从大景别起幅不间断地向小景别落幅变化，使主体与所处环境的联系没有中断，保持了空间的统一性和完整性，具有无可置疑的真实性和可信性。比如从一个舞者表演的全景推到她的手部特写。

                                                            ⑤推镜头推进速度的快慢可以影响和调整画面节奏，从而产生外化的情绪力量。 推镜头使画面框架处于运动之中，直接形成了画面外部的运动节奏。比如，推进的速度缓慢而平稳，便能够表现出安宁、幽静、平和、神秘等氛围；推进的速度急剧而短促，则能显示出一种紧张和不安的气氛，或激动、愤怒等情绪。

                                                            ⑥推镜头可以通过突出一个重要的戏剧元素来表现特定的主题和涵义。 在电影故事片和电视剧中，推镜头将画面从纷乱的场景引到具体的人物，或从人物引到其细小的表情动作等，通过画面语言的独特造型形式强烈地喻示出那些引发情节和事件、烘托情绪和气氛的重要的戏剧元素，从而形成影视剧特有的场面调度和画面语言。

                                                            ⑦推镜头可以加强或减弱运动主体的动感。 如果推镜头构成的画面外部运动与被摄人物画面内的运动形成相对或相反的方向，则会加快或减弱画面的运动感觉。

                                                            # 推镜头拍摄应注意的问题

                                                            • ①推镜头应有其明确的表现意义。
                                                            • ②推镜头的重点是落幅。
                                                            • ③推镜头在推进的过程中，画面构图应始终注意保持主体在画面结构中心的位置。
                                                            • ④推镜头的推进速度要与画面内的情绪和节奏相一致。
                                                            • ⑤在移动机位的推镜头中，画面焦点要随着机位与被摄主体之间距离的变化而变化。

                                                            # 第二节 拉摄（拉镜头）

                                                            拉摄是摄像机逐渐远离被摄主体，或者变动镜头焦距使画面框架由近至远，与主体拉开距离的拍摄方法。用这种方式拍摄的运动画面，称为拉镜头。

                                                            # 拉镜头特征

                                                            ①拉镜头形成视觉后移效果。 在拍摄拉镜头时，镜头向后运动或拉出的过程中，造成画面框架向后运动，使画平面从某一主体开始逐渐退向远方，画面表现出视点后移的效果，呈现一种较小景别向较大景别连续变化的过程，具有小景别转换成大景别的各种特点。

                                                            ②拉镜头将被摄主体由大变小，周围环境由小变大。 拉镜头也分为起幅、拉出、落幅三个部分。镜头拍摄时，画面从某一被摄主体开始，随着镜头向后拉开，被摄主体逐渐远离，在画面中看起来由大变小，而主体周围的环境则不断入画由小变大。

                                                            ③景别由小向大连续变化。 画面从一个女舞者手持的折扇特写拉成人物舞动的中景，再拉开到整个舞台的大场面全景，画面不仅视点后移，而且空间逐渐展开，场面逐渐扩大，呈现一种景别从特写到中景到全景连续变化，而被摄主体则由大变小逐渐远去的过程和效果，是一种后退式的影像句式。

                                                            # 拉镜头功用和表现力

                                                            ①拉镜头有利于表现主体和主体所处环境的关系。 拉镜头在一个连贯的镜头中，既在起幅画面中表明了主体形象，又在落幅画面中表现了主体所处的环境或情境。这种从主体引出环境的表现方式是一种从点到面的表现方式，它在点面关系上具有两个层面的意思：第一，表现某主体在某空间的位置，即此点在此面的位置。拉镜头常有“某人（或某物）在某处”的意味。第二，表明点与面所构成的某种关系。（例子举《焦点访谈》或年北京奥运会或其他自己总结的案例均可）

                                                            ②拉镜头画面的取景范围和表现空间是从小到大不断扩展的，使得画面构图形成多结构化。 （自己总结的课外案例最佳，也可以举《人与自然》或《幸福终点站》）

                                                            ③拉镜头可以通过纵向空间和纵向方位上的画面形象形成对比、反衬或比喻等效果。 （自己总结的课外案例最佳，也可以举《讲述》）

                                                            ④拉镜头从不易推测出整体形象的局部为起幅，有利于调动观众整体形象逐渐出现直至呈现完整形象的想象和猜测。 （自己总结的课外案例最佳，也可以举《僵尸肖恩》）

                                                            ⑤拉镜头在一个镜头中景别连续变化，保持了画面表现时空的完整和连贯。 （自己总结的课外案例最佳，也可以举书中的）

                                                            ⑥拉镜头内部节奏由紧到松，与推镜头相比，较能发挥感情上的余韵，产生许多微妙的感情色彩。 （自己总结的课外案例最佳，也可以举书中的案例）

                                                            ⑦拉镜头常被用作结束性的镜头。拉镜头画面表现的空间的扩展反衬出主体的远离和缩小，从视觉感受上来说，往往有一种退出感、远离感和结束感。 在最终的落幅画面中，主体仿佛像是戏剧舞台上的“退场”和“谢幕”一般。（自己总结的课外案例最佳，也可以举书中的案例）

                                                            ⑧利用拉镜头来作为转场镜头。 从特写拉成全景的拉镜头，由于其起幅特写画面背景空间表现的不确定性，经常在电视剧等节目中被用做转场镜头，它使得场景的转换连贯而不跳跃，流畅而不突兀。（自己总结的课外案例最佳，也可以举书中的案例）

                                                            # 拉镜头拍摄与要求

                                                            # 第三节 摇摄（摇镜头）

                                                            摇摄是指摄像机机位不动，借助于三角架上的活动地盘或者是拍摄者自身的人体，变动摄像机光学镜头轴线的拍摄方法。用摇摄的方式拍摄的画面叫摇镜头。

                                                            # 摇摄特征

                                                            ①摇镜头犹如人们头部环顾四周或将视线由一点移向另一点的视觉效果。（自己总结的课外案例最佳，也可以举书中的案例）
                                                            ②一个完整的摇镜头包括：起幅、摇动、落幅三个相互连贯的部分。（自己总结的课外案例最佳，也可以举书中的案例）
                                                            ③一个摇镜头从起幅到落幅的运动过程，迫使观众不断调整自己的视觉注意力。（自己总结的课外案例最佳，也可以举书中的案例）

                                                            # 摇摄功用和表现力

                                                            ①展示空间，扩大视野。摇镜头通过摄像机的运动将画面向四周扩展，突破了画面框架的空间局限，创造了视觉张力，使画面更加开阔，使周围景物尽收眼底。（自己总结的课外案例最佳，也可以举书中的案例）

                                                            ②有利于通过小景别画面包容更多的视觉信息。（自己总结的课外案例最佳，也可以举书中的案例）

                                                            ③介绍交待同一场景中两个物体的内在联系。将两个物体或事物分别安排在摇镜头的起幅和落幅中，通过镜头摇动将这两点连接起来，这两个物体或事物之间的关系就会被镜头运动形成的连接提示或暗示出来。（自己总结的课外案例最佳，也可以举书中的案例）

                                                            ④利用性质意义相反或相近的两个主体，通过摇镜头把它们连接起来，表示某种暗喻、对比、并列因果关系。通过摇来建立某种对应关系的镜头，如同队列蒙太奇的表现性组接一样，把生活中富有对比因素的两个单独形象连接了起来，使它所表现的意义远远超出了这两个形象本身各自的意义。（自己总结的课外案例最佳，也可以举书中的案例）

                                                            ⑤在表现三个或三个以上主体或主体之间的联系时，摇镜头过时或作减速、或作停顿，以构成一种间歇摇。间歇摇在一个镜头中形成了若干段落或间歇，常常用来表现或揭示一组画面主体由于某一因素或原因所构成的内在联系。这种摇镜头通过镜头的运动轨迹形成了一条无形的线和线上间隔相连的点，把几个主体串联起来，有红线串散珠的艺术效果和个体成整体的集聚作用。（自己总结的课外案例最佳，也可以举书中的案例）

                                                            ⑥在一个稳定的起幅画面后，利用极快的摇摄使画面中的形象全部虚化，以形成具有特殊表现力的甩镜头。将几个起幅清楚稳定而落幅不清楚稳定的急速摇镜头组接起来，从画面效果看酷似急速的间歇摇。（自己总结的课外案例最佳，也可以举书中的案例）

                                                            ⑦用追摇的方式表现运动主体的动态、动势、运动方向和运动轨迹。用长焦距镜头在远处追摇一个运动物体，摇动的方向、角度、速度均以这个被摄运动物体为“基点”：被摄物体朝哪个方向运动，镜头就摇向哪个方向；被摄物体移动快，镜头摇动也快。（自己总结的课外案例最佳，也可以举书中的案例）

                                                            ⑧对一组相同或相似的画面主体用摇的方式让它们逐个出现，可形成一种积累的效果。如同修辞学中的排比句，在一段视觉流中，同形事物的重复出现可以强化人们对这个事物的印象。（自己总结的课外案例最佳，也可以举书中的案例）

                                                            ⑨用摇镜头摇出意外之物，制造悬念，在一个镜头内形成视觉注意力的起伏。（自己总结的课外案例最佳，也可以举书中的案例）

                                                            ⑩利用摇镜头表现一种主观性镜头。在镜头组接中，当前一个镜头表现的是一个人环视四周，那么下一个镜头用摇表现的空间就是前一个镜头中的人所看到的空间。此时摇镜头就表现了戏中人的视线而成为一种主观性镜头。（自己总结的课外案例最佳，也可以举书中的案例）

                                                            11.利用倾斜摇、旋转摇表现特定情绪和气氛。倾斜可以破坏观众欣赏画面时的心理平衡，造成一种动荡感、不稳定感、不安全感。同时，倾斜也可以营造一种欢快、活泼的气氛。倾斜的画面再加上摇动，不稳定、不平衡的感觉便会更加强烈。（自己总结的课外案例最佳，也可以举书中的案例）

                                                            12.巧妙地转场。摇镜头可以通过空间的转换、被摄主体的变换引导观众视线由一处转到另一处，完成观众注意力和兴趣点的转移。

                                                            # 摇摄拍摄与要求

                                                            # 第四节 移摄

                                                            移摄是将摄像机架在活动物体上随之运动而进行的拍摄，用移动摄像的方式拍摄的画面成为移动镜头。

                                                            # 移摄特征

                                                            ①摄像机的运动使得画面框架始终处于运动之中，画面中的物体无论是处于运动状态还是静止状态，都会呈现出位置不断移动的态势。（自己总结的课外案例最佳，也可以举书中的案例）

                                                            ②摄像机的运动，直接调动了观众生活中运动的视觉感受，唤起人们在各种交通工具上及行走时的视觉体验，使观众产生一种身临其境的感受。（自己总结的课外案例最佳，也可以举书中的案例）

                                                            ③移动镜头表现的画面空间是完整而连贯的，摄像机不停地运动，每时每刻都在改变观众的视点，在一个镜头中构成多景别。多构图的造型效果，这就起着一种与蒙太奇相似的作用，最后使镜头有了自身的节奏。（自己总结的课外案例最佳，也可以举书中的案例）

                                                            # 移摄功用和表现力

                                                            ①移动镜头通过摄像机的移动开拓了画面的造型空间，创造出独特的视觉艺术效果。电视艺术是通过电视屏幕表现生活图景的，但是电视画面的表现范围却受到画面四边画框的严格限制，移动摄像则使电视画面造型突破这种限制成为可能。（自己总结的课外案例最佳，也可以举书中的案例）

                                                            ②移动镜头在表现大场面、大纵深、多景物、多层次的复杂场景时，具有气势恢宏的造型效果。移动摄像使摄像机摆脱了机位固定的拍摄方式，可以在所能进入的空间随意运动并通过运动形成的多角度、多景别、多构图画面对一个空间进行立体的、多层次的表现。同时还可以有控制地逐一展开景物，有时只要稍稍改变一下摄像机的位置或角度就能形成一个全新的、引人注目的构图。（自己总结的课外案例最佳，也可以举书中的案例）

                                                            ③移动摄像可以表现某种主观倾向，通过有强烈主观色彩的镜头表现出更为自然生动的真实感和现场感。移动摄像使摄像机成了能动的活跃的物体。机位的运动，直接调动了人们在行进中或运动物体上的视觉感受，我们称之为“主观性镜头”。（自己总结的课外案例最佳，也可以举书中的案例）

                                                            ④移动摄像摆脱定点拍摄后形成多样化的视点，可以表现出各种运动条件下的视觉效果。随着电视技术的不断发展，电视摄录设备日益小型化、轻便化、一体化，移动摄像的形式也越来越丰富，向着多样化、多视点方向发展。（自己总结的课外案例最佳，也可以举书中的案例）

                                                            # 移动镜头的拍摄应注意的问题

                                                            • ①注意保持画面平稳。
                                                            • ②用广焦镜头拍摄会取得较好的画面效果。
                                                            • ③拍摄时应注意随时调整焦点以保证被摄主体始终在景深范围之中。

                                                            # 第五节 跟摄

                                                            跟摄：跟摄是摄像机始终跟随运动的被摄主体一起运动而进行的拍摄。用这种方式拍摄的运动画面，称为跟镜头。

                                                            # 跟摄造型特点

                                                            • ①画面始终跟随一个运动的主体(人物或物体)。
                                                            • ②被摄对象在画框中的位置相对稳定，景别也相对稳定，以使被摄主体的运动保持连贯，进而有利于展示主体在运动中的动态、动姿和动势。
                                                            • ③跟镜头不同于推镜头，也不同于前移动镜头。

                                                            # 跟摄作用

                                                            ①跟镜头能够连续而详尽地表现运动中的被摄主体，它既能突出主体，又能交待主体的运动方向、速度、体态及其与环境的关系。跟镜头是通过摄像机跟随被摄对象一起移动的拍摄方式，它用画框始终“套”住运动中的被摄对象，使被摄对象相对稳定地处于画面的某个位置上，使观众与被摄对象之间的视点相对稳定，形成一种对动态人物或物体的静态表现方式，使动体的运动连贯而清晰地呈现在画面上，以展示人们在动态中的神态变化和性格特点。（自己总结的课外案例最佳，也可以举书中的案例）

                                                            ②跟镜头与被摄对象一起运动，形成一种运动的主体不变静止的背景变化的造型效果，有利于通过人物引出环境。跟镜头的摄像机运动以运动的被摄对象为契机和依据，人物的运动“带”着摄像机的运动，摄像机随着人物将其走过的环境逐一连贯地表现出来。（自己总结的课外案例最佳，也可以举书中的案例）

                                                            ③从人物背后跟随拍摄的跟镜头，由于观众与被摄人物视点的同一，可以表现出一种主观性镜头。摄像机背跟方式的跟镜头，由于观众与被摄人物视点的同一（合一），镜头表现的视向就是被摄人物的视向，因而画面表现的空间也就是被摄人物看到的视觉空间。这种视向的合一，将观众的视点调度到画面内跟着被摄人物走来走去，从而表现出一种强烈的现场感和参与感。（自己总结的课外案例最佳，也可以举书中的案例）

                                                            ④跟镜头对人物、事件、场面的跟随记录的表现方式，在纪实性节目和新闻节目的拍摄中有着重要的纪实性意义。跟镜头的表现方式，不仅使观众置身于事件之中，成为事件的“目击者”，而且还表现出一种客观记录的“姿态”。（自己总结的课外案例最佳，也可以举书中的案例）

                                                            # 跟摄拍摄要求

                                                            # 第六节 升降镜头

                                                            升降拍摄：摄像机借助升降装置等一边升降一边拍摄的方式叫升降拍摄。用这种方式拍摄的运动画面，称为升降镜头。

                                                            # 升降镜头画面造型特点

                                                            ①升降镜头的升降运动带来了画面视域的扩展和收缩。

                                                            ②升降镜头视点的连续变化形成了多角度、多方位的多构图效果。

                                                            # 升降镜头功能和表现力

                                                            ①升降镜头有利于表现高大物体的各个局部。

                                                            ②升降镜头有利于表现纵深空间中的点面关系。

                                                            ③升降镜头常用来展示事件或场面的规模、气势和氛围。

                                                            ④镜头的升降可实现一个镜头内的内容转换与调度。

                                                            ⑤升降镜头可以表现出画面内容中感情状态的变化。

                                                            # 升降镜头注意的问题

                                                            由于升降镜头所带来的视觉感受比较特别，容易今观众感到节目编摄者的主观创作意图，因此，对升降镜头应当慎用，特别是拍摄新闻纪实类节目时尤其需慎重考虑，否则，画面造型的表现性，可能影响节目内容的真实感和客观性。

                                                            # 第七节 综合运动摄像

                                                            综合运动摄像是指摄像机在一个镜头中把推、拉、摇、移、跟、升降等各种运动摄像方式，不同程度地、有机地结合起来的拍摄。用这种方式拍摄得的画面叫综合运动镜头。

                                                            # 综合运动摄像特征

                                                            ①综合运动镜头的镜头综合运动产生了更为复杂多变的画面造型效果。

                                                            ②由镜头的综合运动所形成的电视画面，其运动轨迹是多方向、多方式运动合一后的结果。

                                                            # 综合运动镜头的功用和表现力

                                                            ①综合运动镜头有利于在一个镜头中记录和表现一个场景中一段相对完整的情节。

                                                            ②综合运动镜头是形成电视画面造型形式美的有利手段。

                                                            ③综合运动镜头的连续动态有利于再现现实生活的流程。

                                                            ④综合运动镜头有利于通过画面结构的多元性形成表意方面的多义性。

                                                            ⑤综合运动镜头在较长的连续画面中可以与音乐的旋律变化相互“合拍”，形成画面形象与音乐一体化的节奏感。
